# Tetris

This project is a simple implementation of the Tetris game. The code is based on this [video](https://www.youtube.com/watch?v=wVYKG_ch4yM), 
with some modifications such as using methods from the standard library instead of hardcoding them or changing the logic of some methods.

This project was created using the CLion IDE.
